# Подключение телетайпа Consul 254 к Arduino.

* keyboard - чтение нажатий на клавиатуру и вывод на LCD-дисплей
* print - тестовая программа для вывода "HELLO, WORLD!"
* typewriter - Consul в режиме пишущей машинки
